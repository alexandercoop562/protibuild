# Protibuild a protein simulator and builder

This is an open source app to simulate proteins in an environment and to build new proteins.

## Installation

Before installing, follow the [Bevy setup guide](https://bevy.org/learn/quick-start/getting-started/setup/) to ensure your system has the required dependencies.

Install the `protibuild` binary from GitHub:

```bash
cargo install --git https://github.com/alexandercoop562/protibuild
```

[Documentation](https://protibuild.alexandercoop.com/)
